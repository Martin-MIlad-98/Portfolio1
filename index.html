Featured Projects

Library Management System

I developed a Library Management System using C++ to simulate real-world library operations and strengthen my understanding of core programming concepts. The system enables users to add, search, borrow, and return books through a simple console-based interface. It is designed with a modular structure, ensuring clean organization and easy scalability.

The project focuses on implementing fundamental data structures, including a Linked List for managing the book catalog, a Queue (FIFO) for tracking borrowed books, and a Stack (LIFO) for handling returned books. This experience enhanced my skills in object-oriented programming, problem-solving, and building efficient, well-structured software systems.

Student Grade System

I developed a Student Grade Management System using C++ and the Qt Framework, designed to simulate real-world academic operations through a graphical user interface. The system allows users to manage students, professors, courses, and grades in an organized and user-friendly environment. It provides essential functionalities such as adding and managing records, assigning courses, and handling student grades, all within a structured and interactive GUI. The project emphasizes clear separation between interface and logic, ensuring a clean and maintainable design.

The project strongly demonstrates core Object-Oriented Programming principles, including encapsulation, inheritance, abstraction, and polymorphism. It follows a layered architecture consisting of presentation, control, and core logic layers, enhancing scalability and maintainability. By implementing modular components such as Student, Professor, Course, and GradeManager, I strengthened my skills in designing efficient systems, managing complex relationships, and building well-structured, extensible applications.
